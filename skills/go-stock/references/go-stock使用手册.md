## 1. 软件简介

go-stock 是一款基于大语言模型的 AI 赋能股票分析桌面工具，基于 **Wails + Vue3 + NaiveUI** 技术栈构建，支持 A股、港股、美股的行情数据获取与智能分析。

### 1.1 核心特性

- 多市场行情监控（A股/港股/美股）
- AI 智能分析股票（支持 16 个大模型平台）
- AI 智能体对话（React/PlanExecute 两种模式）
- 浮动 AI 助手（任意页面快速唤起对话）
- 实时 K 线图与分时图（ECharts + TradingView Lightweight Charts）
- 市场资讯聚合（财联社/新浪/外媒/雪球）
- 资金流向与龙虎榜数据
- 涨停梯队与异动监控
- 社区化提示词广场与问答广场
- MCP 服务与技能扩展
- 定时任务与自动预警（5种预警类型）
- 交易日志管理（Beta）
- 弹幕互动社区
- 钉钉推送与本地通知

### 1.2 支持的 AI 模型平台

DeepSeek、硅基流动、智谱AI、字节豆包、阿里云百炼、Moonshot、腾讯混元、讯飞星火、零一万物、MiniMax、百川智能、百度千帆、OpenAI、Azure OpenAI、OpenRouter、Ollama 以及所有兼容 OpenAI API 格式的平台。

### 1.3 数据来源

| 数据来源 | 用途 |
|----------|------|
| 东方财富 | 行情数据、资金流向、财报数据、龙虎榜、K线 |
| 同花顺问财 | 自然语言选股、行情查询、研报搜索 |
| 新浪财经 | 财经新闻、基金数据 |
| 财联社 | 实时市场电报快讯 |
| 雪球 | 热门事件、股票讨论 |
| 华尔街见闻 | 外媒资讯、市场日历 |
| TradingView | 国外财经媒体、K线数据 |

## 2. 安装与启动

### 2.1 下载软件

从 GitHub Releases 页面下载对应版本的安装包：

- Windows 版：`go-stock-windows-amd64.exe`
- macOS 版：`go-stock-darwin-universal`

### 2.2 首次启动

1. 双击运行程序
2. 首次启动会自动初始化股票基础数据（A 股/港股/美股基础信息）
3. 窗口大小会自适应屏幕分辨率
4. 加载完成后进入主界面

### 2.3 数据存储

- 数据库文件：运行目录下 `data/stock.db`（SQLite）
- 字典数据：运行目录下 `data/` 目录
- 配置信息保存在本地数据库中

## 3. 界面布局与导航菜单

### 3.1 整体布局

软件采用**底部水平导航栏 + 上方内容区**的布局方式，导航菜单固定在窗口底部。

### 3.2 导航菜单结构

| 菜单项 | 路由 | 说明 |
|--------|------|------|
| ⭐ 股票自选 | `/` | 自选股管理，含分组子菜单 |
| 📰 市场行情 | `/market` | 市场数据与资讯，含11个子菜单 |
| ✨ 基金自选 | `/fund` | 基金关注与管理（需启用） |
| 🤖 AI智能体 | `/agent` | AI智能体对话（需启用） |
| 🔬 研究中心 | `/research` | 研究分析工具集，含12个子菜单 |
| ⚙️ 设置 | `/settings` | 软件配置 |
| 🐙 关于 | `/about` | 版本信息与赞助 |
| 隐藏至托盘区 | — | 最小化到系统托盘 |
| 退出程序 | — | 关闭应用 |

> ⚠️ 注意：「基金自选」和「AI智能体」两个菜单项需要在后台配置中启用对应功能后才会显示。

### 3.3 窗口标题栏

标题栏动态显示当前市场状态，格式为：

```
go-stock A股交易中|港股休市|美股休市  「投资格言」  [数据来源于网络，仅供参考；投资有风险，入市需谨慎]
```

市场状态每 60 秒自动刷新，同时随机切换一条投资格言。

### 3.4 滚动快讯条

当在设置中启用"显示滚动快讯"后，页面顶部会以跑马灯形式滚动显示市场快讯标签。

### 3.5 水印

软件界面叠加有水印层，用于数据来源声明和防伪标识。

## 4. 股票自选

股票自选是软件的核心页面，用于管理和监控关注的股票。

### 4.1 标签页分组

页面顶部为标签页栏，支持分组管理自选股：

| 操作 | 说明 |
|------|------|
| 点击"全部" | 查看所有自选股 |
| 点击分组标签 | 查看该分组内的股票 |
| 点击"+"按钮 | 新增分组（输入分组名称和排序值） |
| 点击标签关闭按钮 | 删除分组（需二次确认，数据不可恢复） |
| 拖拽标签 | 调整分组排序（"全部"标签不可拖拽） |

分组列表会在启动时从数据库动态加载，每个分组作为"股票自选"菜单的子菜单项显示。

### 4.2 自选股卡片

每只自选股以卡片形式展示，包含以下信息区域：

#### 卡片标题区

- **股票代码标签**：显示股票代码
- **取消关注**按钮：移除该股票
- **AI分析**按钮：发起 AI 智能分析（需启用 AI 功能）
- **移出分组**按钮：将股票从当前分组移出（仅在分组标签下显示）

#### 价格信息区

- **当前价格**：实时价格，涨为红色，跌为绿色，平为默认色
- **盘前盘后价格**：仅美股显示盘前盘后价格及涨跌幅
- **涨跌幅百分比**：带数字动画效果
- **今日盈亏金额**：仅在设置了成本价和持仓数量后显示
- **迷你走势线**：仅在分组标签下显示

#### 行情数据区

四宫格显示：最高价+最高涨幅百分比、最低价+最低跌幅百分比、昨日收盘价、今日开盘价

#### 盘口数据区（仅A股）

可折叠面板，展示买卖五档盘口数据。折叠时显示买一报价(申报量)/卖一报价(申报量)，展开后显示买一至买五、卖一至卖五各档的报价和申报量。

#### 卡片底部

- 日期和时间
- 持股数量标签（设置持仓后显示）
- 成本盈亏标签："成本:xxx*xxx 盈亏% (xxx ¥)"
- **多周期K线**按钮

#### 卡片操作按钮

| 按钮 | 功能 |
|------|------|
| 成本 | 打开成本设置对话框 |
| 分时 | 打开分时走势图 |
| 日K | 打开日 K 线图 |
| 资金 | 打开资金趋势图（仅A股显示） |
| 详情 | 跳转问财搜索页面 |
| 公告 | 跳转公司公告页面（仅A股） |
| 研报 | 跳转个股研报页面（仅A股） |
| 设置分组 | 下拉菜单，将股票添加到指定分组 |

### 4.3 底部搜索栏

位于页面右下角，包含：

- **自动完成输入框**：输入股票名称/代码/指数关键词，实时模糊匹配
- **关注**按钮：将搜索到的股票添加到自选列表
- **发送弹幕**按钮（需启用弹幕功能）：通过 WebSocket 发送弹幕消息

> 💡 提示：搜索已关注的股票时，页面会自动滚动到对应卡片并闪烁红色边框5秒。

### 4.4 成本设置对话框

点击"成本"按钮打开，宽度800px，表单字段：

| 字段 | 说明 |
|------|------|
| 股票成本 | 成本价，支持 HK$/¥ 后缀自动适配 |
| 股票数量 | 持仓数量，步进100 |
| 涨跌提醒 | 涨跌幅预警百分比 |
| 股价提醒 | 绝对价格预警 |
| 开仓价 | 开仓价位 |
| 股票排序 | 排序值 |
| 止盈价 | 止盈价格 |
| 止损价 | 止损价格 |

### 4.5 分时图对话框

点击"分时"按钮打开，宽度1000px：

- 使用 ECharts 绘制当日分时走势图
- 包含股价线和成交量柱状图
- 标记最高/最低价位点
- 标记均价线、开盘价线、收盘价线
- 低于开盘价（绿色）、大于开盘价小于收盘价（蓝色）、大于收盘价（红色）分段着色
- **自动刷新**：打开后每 10 秒自动刷新

### 4.6 日K线图对话框

点击"日K"按钮打开，宽度1000px：

- 日K蜡烛图
- MA5/MA10/MA20/MA30 均线
- 成交量柱状图
- 数据缩放滑块，默认显示近14%的数据
- 标记最高/最低/平均收盘价
- 鼠标悬停提示框（开盘/收盘/最低/最高/成交量/各均线值）
- 数据来源：近365天K线数据

### 4.7 AI分析对话框

点击"AI分析"按钮打开，宽度800px：

- Markdown 格式渲染 AI 分析结果，加载状态旋转动画，导出 PDF 工具栏
- 底部信息：模型名称标签 + 分析时间 + 风险提示
- 工具调用开关 / 思考模式开关
- 三个下拉选择框：AI 模型服务配置、系统提示词、用户提示词
- 问题输入框：自定义提问内容

| 按钮 | 功能 |
|------|------|
| 开始AI分析 | 重新发起 AI 流式分析 |
| 保存为图片 | 截图保存为 PNG |
| 复制到剪切板 | 复制分析结果文本 |
| 保存为Markdown | 保存为 `.md` 文件 |
| 保存为Word | 导出为 `.docx` 文件 |
| 分享到项目社区 | 分享到社区 |

### 4.8 资金趋势对话框

点击"资金"按钮打开，宽度1000px，使用 MoneyTrend 组件展示近 360 天资金趋势图，图表高度 500px。

### 4.9 多周期K线对话框

点击"多周期K线"按钮打开，最大宽度 1100px：

- 使用轻量级 K 线图组件（TradingView Lightweight Charts）
- 支持拖拽调整开仓价/止损价/止盈价/成本价线，调整后自动保存

> ⚠️ 注意：VIP权限控制：VIP2 及以上用户可正常使用，权限不足时提示并自动关闭。美股代码（`gb_` 前缀）不支持多周期K线。

### 4.10 价位线预警系统

每次价格更新时自动检查：

| 预警类型 | 触发条件 | 推送类型 |
|----------|----------|----------|
| 涨跌幅预警 | 涨跌幅达到设定百分比 | 类型1 |
| 股价预警 | 股价触及设定价格 | 类型2 |
| 成本价预警 | 成本盈亏达到设定值 | 类型3 |
| 止盈触及 | 当前价 ≥ 止盈价 | 类型4 |
| 止损触及 | 当前价 ≤ 止损价 | 类型5 |

防抖机制：同一价格不重复提醒，两次通知间隔至少60秒。

### 4.11 弹幕系统

- 连接 WebSocket 服务器实时接收弹幕
- 使用 vue3-danmaku 组件展示弹幕飘过效果
- 可在设置中开启/关闭弹幕功能

## 5. 市场行情

市场行情页面包含 11 个子标签页，覆盖全方位市场数据。

### 5.1 市场快讯

**24小时热词分析**：顶部嵌入热词分析图表，展示最近24小时市场热词分布。

**三栏新闻列表**：

| 栏目 | 数据来源 | 说明 |
|------|----------|------|
| 财联社电报 | 财联社 | 实时市场快讯，红色标题为重磅消息 |

## 6. 基金自选

基金自选页面用于关注和管理基金产品。

> ⚠️ 注意：此功能需启用"基金自选"后，菜单才会显示。

### 6.1 基金卡片

3列网格布局，每只基金展示为一张卡片：

- **标题**：基金名称（超长自动省略）
- **代码标签**：基金代码
- **取消关注**标签
- **估算净值** + **估算涨幅**：正数红色，负数绿色
- **单位净值** + 日期
- **涨幅标签组**：近一月/三月/六月/一年/三年/五年/今年来/成立来
- **底部信息**：基金类型、基金公司+基金经理

| 按钮 | 功能 |
|------|------|
| 走势 | 打开走势图弹窗（展示新浪基金走势GIF图） |
| 详情 | 打开东方财富基金详情页 |

### 6.2 底部搜索栏

- 自动完成输入框：输入基金名称/代码搜索
- "关注"按钮：将基金添加到关注列表
- "发送弹幕"按钮：发送弹幕消息

### 6.3 数据刷新

每 60 秒自动刷新关注列表数据。

## 7. AI智能体

AI 智能体页面提供对话式交互体验，支持两种 Agent 模式。

> ⚠️ 注意：此功能需启用"AI智能体"后，菜单才会显示，同时需在设置中开启"AI诊股总开关"并配置 AI 模型。

### 7.1 界面布局

- **聊天区域**：主体消息列表，支持滚动
- **执行步骤区域**：AI 使用工具时，显示步骤列表和状态指示（已完成=绿色、工具调用=蓝色、活跃=橙色）
- **推理过程区域**：展开/收起 AI 的思考过程
- **分析报告区域**：可折叠的分析报告区域，AI 生成的结构化报告
- **消息操作**：每条消息支持复制操作

### 7.2 底部输入区

| 元素 | 说明 |
|------|------|
| AI配置选择器 | 选择不同的 AI 模型配置 |
| Agent模式选择器 | 三种模式（见7.3） |
| 输入框 | 输入消息内容 |
| 发送按钮 | 发送消息 |

### 7.3 Agent 模式

| 模式 | 标识 | 说明 |
|------|------|------|
| 🤖 自动 | 自动 | 系统根据问题复杂度自动选择模式 |
| ⚡ 快速 | react | 适合简单查询，单步工具调用即可回答 |
| 🧠 规划 | plan_execute | 适合复杂分析，AI 先制定计划后逐步执行 |

### 7.4 AI 工具调用

AI 智能体在分析时可以自动调用 80+ 种工具获取实时数据，按8大类别组织：

| 类别 | 包含工具举例 |
|------|-------------|
| base 基础工具 | 当前时间、交易日判断、节假日查询、交易日历 |
| stock_analysis 股票分析 | K线数据、财务信息、股东户数、融资融券、公司信息、分红配送 |
| market 市场数据 | 全球股指、行业排名、经济数据(GDP/CPI/PPI/PMI)、市场统计 |
| screening 选股筛选 | 指标选股、自然语言选股、ETF筛选、板块选股 |
| money_flow 资金流向 | 资金趋势、主力散户排名、龙虎榜、行业资金排名 |
| news_research 新闻研究 | 财联社电报、新浪财经、研报、公告、华尔街见闻 |
| ai_analysis AI分析 | 可比公司分析、财务问答、热点发现、涨停梯队 |
| operations 运营操作 | 钉钉推送、AI推荐股票、设置交易价位 |

系统根据用户问题的关键词自动匹配对应的工具分组，同时可动态加载 MCP 工具和根据技能触发关键词注入技能提示词。

### 7.5 流式响应

- AI 回复以流式方式逐字展示
- 支持 `[STEP]` 标记的执行步骤展示
- 支持工具调用信息展示（函数名+参数）
- 自动检测内容中的 JSON 并提取为"分析报告"
- 每 1.5 秒定时格式化 Markdown

### 7.6 PlanExecute 模式详解

PlanExecute 模式采用 Planner→Executor→Replanner 管道，最大迭代7次。AI 首先制定分析计划，然后逐步执行每个步骤，执行完成后重新评估是否需要调整计划。系统对执行结果进行智能压缩：最近步骤获得更多预算，较早步骤预算减少，确保关键信息不丢失。

### 7.7 其他交互

- **回到底部**：悬浮按钮，滚动时自动出现
- **停止生成**：中断 AI 回复
- **复制操作**：复制消息内容
- **暗黑主题**：自动跟随系统配置

### 12.2 常见问题

#### Q: AI 分析不工作？

A: 检查以下几项：
1. 设置中是否已启用"AI诊股总开关"
2. 是否正确配置了 AI 模型的 API Key
3. 网络连接是否正常
4. 如使用代理，确认代理设置正确

#### Q: 数据不更新？

A: 尝试以下方法：
1. 检查网络连接
2. 确认当前是否为交易时间（休市期间部分数据不会刷新）
3. 重启软件
4. 在设置中启用"启动时更新基础信息"

#### Q: 如何添加自选股？

A: 在股票自选页面右下角的搜索栏中输入股票名称或代码，选中后点击"关注"按钮。

#### Q: VIP 功能无法使用？

A: 确认以下几项：
1. 赞助码是否正确输入
2. 点击"验证"按钮确认赞助码有效
3. 重启软件使 VIP 状态生效

#### Q: 如何收到股票预警通知？

A: 需要完成以下配置：
1. 在成本设置中设置涨跌提醒百分比或股价提醒
2. 在设置中开启"本地推送"或配置"钉钉推送"
3. 确保在交易时间内运行软件

#### Q: AI 智能体菜单不显示？

A: AI 智能体菜单需要在后台配置中启用对应功能后才会显示。

#### Q: 多周期K线提示权限不足？

A: 多周期K线功能需要 VIP2 及以上等级。请在关于页面查看当前 VIP 等级，并在设置中输入赞助码升级。

#### Q: 浮动AI助手无法使用？

A: 浮动AI助手需要 VIP2 及以上赞助用户才能使用。请在设置中输入赞助码升级到VIP2。

#### Q: 提示词广场/问答广场无法访问？

A: 请检查设置中的"提示词广场地址"是否正确，默认为 http://go-stock.sparkmemory.top:1918/api。如网络受限可尝试修改为其他可用地址。

#### Q: 问财功能不可用？

A: 问财功能需要配置"问财API密钥"，请在设置中填写同花顺问财开放平台的 API Key，配置后可使用问财智能选股、行情查询、研报搜索等功能。

## 13. 技术架构

### 13.1 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue3 + NaiveUI + ECharts + TradingView Lightweight Charts + vue3-danmaku + md-editor-v3 |
| 后端 | Go + Wails 框架 |
| 数据库 | SQLite + GORM ORM |
| AI框架 | cloudwego eino（React Agent + PlanExecute Agent） |

### 13.2 AI Agent 架构

基于 cloudwego eino 框架构建。提供两种 Agent 实现：

- **React Agent**：简单查询模式，单步工具调用，包含错误恢复和未知工具处理中间件，以及智能内容压缩（优先级：标题 > 摘要 > 数据 > 其他）
- **PlanExecute Agent**：复杂分析模式，Planner→Executor→Replanner 管道，最大7次迭代，分级压缩执行结果

### 13.3 MCP 与技能扩展

**MCP（Model Context Protocol）**：在研究中心添加 MCP 服务器配置后，AI 智能体运行时自动通过 `getMCPTools()` 动态加载外部工具。

**技能系统**：通过技能管理器创建自定义技能，配置触发关键词和绑定MCP服务。AI 智能体运行时通过 `buildSkillPrompt()` 检测提问内容是否匹配技能触发关键词，匹配则注入技能系统提示词。技能分类包括：股票分析、技术分析、基本面分析、量化策略、风险管理、资讯研究、通用。

### 13.4 错误上报

前端内置 `window.onerror` 处理器，会将 JavaScript 错误信息（页面、消息、来源、行号、列号、堆栈）自动发送给后端 EventsEmit，便于问题排查。
